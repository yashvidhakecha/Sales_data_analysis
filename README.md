# Sales Data Analysis

This project was completed for the Programming for Data Analytics module.

The analysis was performed using Python in Google Colab.

## Dataset
Sales Data_PDA_4052.xlsx

## Tools Used
Python
Pandas
Matplotlib
Seaborn

## Analysis Performed
- Data inspection and cleaning
- Missing value check
- Sales performance analysis by salesperson
- Priority vs sales value analysis
- Sales trend over time visualization
- Correlation analysis

## Files in Repository
sales_Data_analysis.ipynb – Python analysis notebook  
Sales Data_PDA_4052.xlsx – Dataset used for analysis
